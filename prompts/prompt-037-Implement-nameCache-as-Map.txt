Enhance the XName class:
- Currently, the nameCache is implemented as an array. Therefore, looking up a name object by sequentially searching through the array will be inefficient.
- Instead, replace nameCache with a Map<string, XNameCacheEntry> object. The string is the fully qualified name in clark notation, and when looking up the XNameCacheEntry object, it will be more efficient to retrieve it from a Map<string, XNameCacheEntry>.
